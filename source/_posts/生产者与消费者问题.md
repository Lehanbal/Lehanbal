## `synchronized`实现

资源类：使用了一个List队列来当作锁的对象

```java
package top.lehanbal.ProducerAndConsumer;

import java.util.LinkedList;
import java.util.List;

public class Resource {
    final int max_size = 5;

    final List<Object> list = new LinkedList<>();

    public void production() {
        synchronized (list) {
            while (list.size() + 1 > max_size) {
                try {
                    System.out.println("仓库满了，生产者暂停");
                    list.wait();
                } catch (InterruptedException e) {
                    e.printStackTrace();
                }
            }
            list.add(new Object());
            System.out.println("生产者生产了一个产品，当前产品剩余数量为" + list.size());
            list.notifyAll();
        }

    }

    public void consume() {
        synchronized (list) {
            while (list.size() == 0) {
                try {
                    System.out.println("仓库空了，消费者暂停");
                    list.wait();
                } catch (InterruptedException e) {
                    e.printStackTrace();
                }
            }
            list.remove(list.size() - 1);
            System.out.println("消费者消费了一个产品，当前产品剩余数量为" + list.size());
            list.notifyAll();
        }

    }
}
```

## `ReentrankLock`实现

与synchronized实现的方式很像，不过是将synchronized代码块包裹的地方换成使用lock包裹罢了。

资源类：

```java
package top.lehanbal.ProducerAndConsumer;

import java.util.LinkedList;
import java.util.List;
import java.util.concurrent.locks.Condition;
import java.util.concurrent.locks.ReentrantLock;

public class Resource {
    final int max_size = 5;

    final List<Object> list = new LinkedList<>();
    ReentrantLock lock = new ReentrantLock();

    private final Condition empty = lock.newCondition();
    private final Condition full = lock.newCondition();

    public void production() {
        lock.lock();
        try{
            while (list.size() + 1 > max_size) {
                try {
                    System.out.println("仓库满了，生产者暂停");
                    full.await();
                } catch (InterruptedException e) {
                    e.printStackTrace();
                }
            }
            list.add(new Object());
            System.out.println("生产者生产了一个产品，当前产品剩余数量为" + list.size());
            empty.signalAll();
        }finally {
            lock.unlock();
        }
    }

    public void consume() {
        lock.lock();
        try{
            while (list.size() == 0) {
                try {
                    System.out.println("仓库空了，消费者暂停");
                    empty.await();
                } catch (InterruptedException e) {
                    e.printStackTrace();
                }
            }
            list.remove(list.size() - 1);
            System.out.println("消费者消费了一个产品，当前产品剩余数量为" + list.size());
            full.signalAll();
        }finally {
            lock.unlock();
        }
    }
}
```
